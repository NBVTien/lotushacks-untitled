# Future Feature Ideas

Ideas for post-MVP development. Add new ideas here as they come up.

---

## Candidate Experience

- [ ] AI interview chatbot — auto-screen candidates with conversational AI
- [ ] Candidate status tracking — let candidates see where they are in the pipeline
- [ ] Multi-file upload — support DOCX, images, portfolio links alongside PDF
- [ ] Candidate self-enrichment — let candidates connect GitHub/LinkedIn directly (OAuth)

## Recruiter Tools

- [ ] Bulk CV upload — drag & drop multiple PDFs at once
- [ ] Candidate comparison — side-by-side view of 2-3 candidates
- [ ] Shortlist / pipeline stages — move candidates through hiring funnel (screening → interview → offer)
- [ ] Email templates — send rejection/interview invite directly from dashboard
- [ ] Notes & comments — recruiters can annotate candidates
- [ ] Team collaboration — multiple recruiters can review and comment

## AI & Scoring

- [ ] Custom scoring weights — recruiter adjusts importance of skills vs experience vs GitHub
- [ ] Re-score on demand — re-run AI scoring after enrichment data updates
- [ ] Bias detection — flag potential bias in scoring explanations
- [ ] Skill gap analysis — suggest training/upskilling for partial matches
- [ ] Portfolio analysis — crawl personal websites and extract project details
- [ ] Stack Overflow / dev.to enrichment — analyze Q&A and blog contributions

## Analytics & Insights

- [ ] Hiring funnel dashboard — conversion rates at each stage
- [ ] Time-to-hire metrics — track how long from upload to decision
- [ ] Score distribution charts — visualize candidate quality per job
- [ ] Source quality analysis — which channels produce best candidates
- [ ] Diversity metrics — anonymized demographic insights

## Integration

- [ ] ATS integration — sync with Greenhouse, Lever, Workable
- [ ] Slack notifications — alert when new candidate scored or high match found
- [ ] Calendar integration — schedule interviews directly
- [ ] Job board syndication — post jobs to LinkedIn, Indeed, etc.
- [ ] Webhook events — notify external systems on candidate status changes

## Platform

- [ ] Multi-language support — Vietnamese, English, Japanese
- [ ] Role-based access — admin, recruiter, hiring manager, viewer
- [ ] Audit log — track all actions for compliance
- [ ] SSO / OAuth login — Google, Microsoft, SAML
- [ ] API rate limiting & usage quotas
- [ ] Tenant isolation — full multi-tenancy with data separation
